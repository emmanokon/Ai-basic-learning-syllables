60-Hour AI Training Syllabus

Created by: [Emmanuel Okon]
Date: June 26, 2025
Duration: 60 hours (12 weeks, 5 hours/week)
Prerequisites: Basic Python programming, high school-level math (algebra, statistics)

Course Overview

This 60-hour course introduces Artificial Intelligence (AI) concepts, tools, and applications. It covers Machine Learning (ML), Deep Learning (DL), Natural Language Processing (NLP), Reinforcement Learning (RL), and AI ethics, with hands-on projects to build practical skills.

Learning Outcomes

  • Understand AI, ML, DL, NLP, and RL concepts and applications.
  • Gain proficiency in Python-based AI tools (scikit-learn, TensorFlow, PyTorch).
  • Preprocess data, train models, and deploy AI solutions.
  • Address ethical challenges in AI.
  • Complete a capstone project.

Weekly Breakdown

Week Objective Topics (Theory + Practical) Activities Resources
1 Understand AI fundamentals What is AI? Narrow, General, Super AI.
History: From Turing to modern AI.
AI vs. ML vs. DL.
Applications: Healthcare, finance, gaming.
Ethical considerations (3h theory, 2h activities). 		Case study: Analyze AI applications.
Discussion: Ethical dilemmas (bias, privacy). 		Articles, videos, slides
2 Grasp ML basics ML paradigms: Supervised, unsupervised, RL.
Key concepts: Features, labels, train-test split.
Algorithms: Linear regression, logistic regression, decision trees.
Metrics: Accuracy, precision, recall (3h theory, 2h practical). 		Install Python, Jupyter, NumPy, pandas.
Implement linear regression with scikit-learn. 		Python, scikit-learn, Google Colab
3 Prepare and analyze data Data quality importance.
Preprocessing: Cleaning, normalization, missing values.
EDA: Visualizations, correlations, outliers.
Tools: Pandas, Matplotlib, Seaborn (2h theory, 3h practical). 		EDA on Titanic dataset.
Clean dataset and visualize insights. 		Kaggle datasets, pandas tutorials
4 Dive into supervised learning Algorithms: SVM, k-NN, Random Forests.
Overfitting, underfitting, bias-variance tradeoff.
Hyperparameter tuning, cross-validation (3h theory, 2h practical). 		Build and tune Random Forest model.
Compare algorithm performance. 		Scikit-learn, Kaggle kernels
5 Understand unsupervised learning Clustering and dimensionality reduction.
Algorithms: k-Means, hierarchical clustering, PCA.
Applications: Customer segmentation, anomaly detection (3h theory, 2h practical). 		Apply k-Means to customer data.
Visualize clusters. 		Scikit-learn, UCI ML Repository
6 Explore neural networks Neural networks: Perceptrons, layers, activation functions.
Frameworks: TensorFlow, PyTorch.
Backpropagation, gradient descent.
Introduction to CNNs (3h theory, 2h practical). 		Build neural network with TensorFlow/Keras.
Experiment with activation functions. 		TensorFlow, PyTorch tutorials
7 Apply DL to computer vision CNN architecture: Convolution, pooling, fully connected layers.
Image preprocessing, augmentation.
Applications: Image classification, object detection (2h theory, 3h practical). 		Train CNN on CIFAR-10 dataset.
Explore pre-trained models (VGG, ResNet). 		Keras, TensorFlow Hub, Kaggle datasets
8 Understand NLP NLP pipeline: Tokenization, stemming, lemmatization, embeddings.
Tasks: Sentiment analysis, text classification.
Word embeddings: Word2Vec, GloVe (3h theory, 2h practical). 		Sentiment analysis on IMDB reviews.
Experiment with embeddings using Gensim. 		NLTK, spaCy, Hugging Face
9 Explore transformers Transformers: Attention, BERT, GPT.
Fine-tuning pre-trained models.
Applications: Chatbots, text generation, translation (2h theory, 3h practical). 		Fine-tune BERT for text classification.
Build simple chatbot. 		Hugging Face, Google Colab Pro
10 Introduce RL concepts RL framework: Agents, environments, rewards.
Algorithms: Q-Learning, DQN.
Applications: Game playing, robotics (3h theory, 2h practical). 		Implement Q-Learning in OpenAI Gym.
Simulate RL agent in grid-world. 		OpenAI Gym, RL tutorials
11 Deploy models and address ethics Deployment: APIs, cloud platforms (AWS, Azure).
Model monitoring, maintenance.
Ethics: Bias, fairness, transparency.
Regulations: GDPR, AI governance (3h theory, 2h activities). 		Deploy ML model with Flask/FastAPI.
Discuss AI ethics case studies. 		Flask, FastAPI, ethics case studies
12 Apply knowledge to a project Review: ML, DL, NLP, RL, ethics.
Capstone: Build end-to-end AI solution (e.g., image classifier, chatbot) (1h review, 4h project). 		Work on project (team or individual).
Present project outcomes. 		Open datasets, cloud platforms

Additional Details

  • Assessment: Weekly assignments, capstone project, participation.
  • Tools: Python, Jupyter, scikit-learn, TensorFlow, PyTorch, Hugging Face, Flask/FastAPI.
  • Format: 60% theory (36h), 40% practical (24h).
  • Notes:
    • Adaptable for shorter/longer formats.
    • Advanced topics (e.g., GANs) can be added for experienced learners.
    • Use cloud platforms (e.g., Google Colab Pro) for compute-intensive tasks.
